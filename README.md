# 飞书侧边栏签名插件

这是根据 `signature-service` 后端接口重写的飞书多维表格侧边栏前端。

## 后端地址

默认连接：

```text
https://liupanqing.asia
```

## 功能

- 读取当前多维表格和字段列表
- 选择需要展示给签字人的字段
- 选择签名链接、签字状态、签名截图链接字段
- 调用服务器创建配置和签名任务
- 为当前记录、手动选择的记录或当前视图可见记录生成签名链接
- 将签名链接和待签字状态写回飞书表格

## 使用

```bash
npm install
npm run build
```

构建完成后，将 `dist` 目录按飞书插件要求上传。

## 说明

页面中填写的“多维表格授权码”会传给后端，用于后端读取记录、下载附件、写回签名状态。授权码应由多维表格管理员/所有者在飞书侧边栏插件中获取。
